# SEO Wordbook / Keyword Library

This document is a reusable keyword + copy library for improving organic search visibility for the static pages in this repo.

## How to use this

- Treat each section as a “pick list” when writing:
  - Page titles
  - Meta descriptions
  - H1/H2 headings
  - On-page copy blocks
  - FAQ sections
  - Internal link anchor text
  - Image `alt` text
- Prefer wording that is:
  - Specific
  - Accurate
  - Non-misleading
  - Consistent with what the business can actually deliver

## “Avoid / be careful” terms

Use these cautiously to reduce ad/SEO compliance risk and avoid misleading content.

- guaranteed mortgage approval
- guaranteed valuation increase
- instant certification
- lender-approved (only if you can substantiate and name the basis)
- “worthless home” / scare language
- claims about fire hazard without evidence

## Page-title templates

Keep titles concise. Aim for ~50–60 characters when possible.

- Spray Foam Removal Assessment | {Brand}
- Spray Foam Removal Specialists | {Brand}
- Spray Foam Mortgage Issues Help | {Brand}
- Spray Foam Removal Quote & Assessment | {Brand}
- Remove Loft Spray Foam Safely | {Brand}

### Title variants with location

- Spray Foam Removal in {Location} | {Brand}
- Spray Foam Removal Assessment ({Location}) | {Brand}

## Meta-description templates

Aim for ~140–160 characters. Mention service area and next step.

- Spray foam flagged by a surveyor or lender? Request an assessment and removal quote. Serving England, Wales & Scotland. Call-back available.
- Professional spray foam removal and remediation support. Get clear next steps, realistic costs and timelines. England, Wales & Scotland.
- Selling or remortgaging with spray foam? We assess what’s installed and provide a removal plan and quote. Book an assessment today.

## Basic on-page checklist (quick)

- One clear H1 per page
- Use the primary keyword in:
  - Title
  - H1
  - First paragraph
- Use 2–4 related phrases from the same cluster (avoid keyword stuffing)
- Add 4–10 FAQs if the page is intended to rank organically
- Ensure internal links between:
  - Demo -> landing
  - Landing -> main site
